Giới Thiệu Bản Thân

Xin chào! Tôi là Nguyễn Quang Hùng, một sinh viên ngành Kỹ thuật phần mềm tại Đại học Công nghệ TP.HCM. Tôi yêu thích việc học hỏi và thực hành các công nghệ mới. Tôi đã từng tiếp xúc qua ngôn ngữ lập trình Java và JavaScript nên tôi luôn muốn chia sẻ những gì tôi đã tiếp xúc được đến cho mọi người biết cũng như những trải nghiệm tôi đạt được.

Kinh Nghiệm

  • Phát triển ứng dụng web với Spring Boot, tập trung vào việc xây dựng các API RESTful và tích hợp các công nghệ như Hibernate, Spring Data JPA, và Spring Security.
  • Thiết kế và triển khai hệ thống API REST để hỗ trợ giao tiếp giữa frontend và backend, đảm bảo hiệu suất cao và dễ dàng mở rộng.
  • Phát triển ứng dụng di động bằng Flutter, bao gồm giao diện người dùng tùy chỉnh và tích hợp các dịch vụ backend thông qua API.
  • Xây dựng các dự án sử dụng Java và JavaScript, bao gồm ứng dụng quản lý, thương mại điện tử và hệ thống quản lý nội dung.
  • Kinh nghiệm làm việc với các cơ sở dữ liệu như MySQL, PostgreSQL, và MongoDB, tối ưu hóa truy vấn và thiết kế mô hình dữ liệu.
  • Thành thạo việc triển khai ứng dụng trên các môi trường khác nhau, bao gồm server nội bộ và nền tảng đám mây (AWS, Heroku).
  • Kỹ năng xử lý lỗi và gỡ rối hiệu quả, cải thiện tính ổn định và bảo mật của hệ thống.
  • Hợp tác trong nhóm phát triển Agile, sử dụng các công cụ quản lý dự án như JIRA và Git để đảm bảo tiến độ và chất lượng dự án.
