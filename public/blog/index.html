Hướng Dẫn JavaScript Cơ Bản

JavaScript là ngôn ngữ lập trình phổ biến trên web, dùng để tạo các trang web tương tác. Bài viết này sẽ hướng dẫn bạn các khái niệm cơ bản về JavaScript cùng với ví dụ minh họa.

  1. Khai báo biến Trong JavaScript, bạn có thể khai báo biến bằng var, let, hoặc const.
// Sử dụng var (ít dùng trong ES6 trở đi)
var x = 10;

// Sử dụng let (dùng khi giá trị biến có thể thay đổi)
let y = 20;

// Sử dụng const (dùng cho giá trị không thay đổi)
const z = 30;

console.log(x, y, z);
``` // Kết quả: 10 20 30
2. Câu lệnh điều kiện
Câu lệnh điều kiện (if, else if, else) giúp kiểm tra điều kiện và thực hiện hành động tương ứng.

```javascript
let age = 18;

if (age < 18) {
  console.log("Bạn chưa đủ tuổi.");
} else if (age === 18) {
  console.log("Chúc mừng bạn vừa đủ tuổi!");
} else {
  console.log("Bạn đã đủ tuổi.");
}
  1. Vòng lặp Vòng lặp dùng để thực hiện một hành động nhiều lần.

Hướng Dẫn Java Cơ bản

Giới Thiệu Về Lập Trình Java Java là một ngôn ngữ lập trình mạnh mẽ, phổ biến và được sử dụng rộng rãi trong phát triển phần mềm, đặc biệt là trong các ứng dụng di động (Android), ứng dụng web, và hệ thống lớn. Một trong những điểm mạnh của Java là tính khả chuyển (portability) nhờ vào triết lý “viết một lần, chạy mọi nơi” với nền tảng Java Virtual Machine (JVM).
